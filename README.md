とかの場合　人数分のボタンで量が解れば便利なのに　と思い作ってみました

私はグラム変換して作っているので　大さじ小さじは考えていません

粉物の大さじは　15gではなく　14gだ　とかは無視です
小さじ　1/2 とか量れている人います？　2.5gですよ
