料理の作り方ではなく　素材に特化したアプリです
1人前のレシピはあっても3人前の場合は全て電卓や頭で3倍しなければなりません

プリンを複数個作る時に　「牛乳1Lあるから何個作れるかな」とか「玉子5個あるから・・」
とかの場合　人数分のボタンで量が解れば便利なのに　と思い作ってみました

大さじ小さじは考えていません
大さじ　15g 　小さじ　5g　 一合　150g　で考えています
小数点以下は考えていません　

そんな感じで　ゆるーく目安のアプリです
